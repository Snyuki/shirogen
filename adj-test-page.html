German

[ edit ]

Etymology

[ edit ]

abhängen +‎ -ig

Pronunciation

[ edit ]
  • IPA ( key ) : /ˈaphɛŋɪç/ ( standard )
  • IPA ( key ) : /ˈaphɛŋɪk/ ( common form in southern Germany, Austria, and Switzerland )
  • Audio : ( file )
  • Hyphenation: ab‧hän‧gig

Adjective

[ edit ]

abhängig ( strong nominative masculine singular abhängiger , comparative abhängiger , superlative am abhängigsten )

  1. dependent [ with von ‘on something’ ]
    Synonym: angewiesen (auf)
  2. addicted [ with von ‘to something’ ]
    Synonym: süchtig (nach)
